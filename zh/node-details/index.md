## 📊 数据输入节点

### 价格数据
- **[Binance Price Node](binance-price-node.md)** - 获取 Binance 交易所的 K 线数据和实时价格

### 新闻聚合（即将推出）
- **新闻聚合节点** - 获取最新行业资讯

### 数据存储
- **[Dataset Input Node](dataset-input-node.md)** - 从 Google Sheets 读取数据

### 区块链数据（即将推出）
- **链上数据节点** - 读取区块链上的数据
